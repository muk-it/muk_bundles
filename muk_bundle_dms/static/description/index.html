Documents Bundle

Document Management System

MuK IT GmbH - www.mukit.at

Overview

MuK Documents is a module to create, manage and view files directly within Odoo. This bundle includes extensions to release the full potential and fully integrate the Documents app into Odoo.

Key Features

Documents Access

MuK Documents Access extends the document module by an extended access control. It allows you to create and manage access groups. These groups can be used to specify access to documents.

Documents Widgets

This module allows the user to use DMS documents as input for any binary field. Both ImageBinary and FileBinary are supported. Additional buttons are available to select a file. The content of the file is copied afterwards so there is no link between file and binary field.

Documents File

MuK Documents File enables a new save option. Files saved with this option will be stored inside a file storage. The file store is not synronized on both sides. So the saved files should not be changed manually, as this can lead to unexpected system states.

Documents Share

Extends the Share Dialog to support sharing files. Files which are shared with other users can be previewed directly inside the Chatter. Go to any form view and in readonly mode there is a share button on the right side of the form sheet. By clicking on this button the share dialog opens and the currently active form can be shared with another user or into a channel.

Documents Finder

MuK Documents finder gives you a file browser like you are used to in windows, mac or linux operating systems. You can browse your files, copy and zip them and even perform simple text editing tasks.

Documents Attachment

MuK Documents Attachment allows to store Odoo Attachments inside the Document Management System. Attachments stored in the document management system are still available as attachments in Odoo. As with all other files, the storage type can be set via the setttings.

Documents Attachment Wizard

This module allows the user to use DMS documents as attachments. This can be used, for example, to append a document to a record.In the settings a flag can be set which specifies whether files should be copied or linked. If the flag is active, files are copied. Copied files are separate and changing or deleting the file does not change the attachment. In the other case files are linked and changing or deleting the file does change or delete the attachment.

Documents Attachment Rules

MuK Documents Attachment allows to store Odoo Attachments inside the Document Management System. By creating rules attachments can be placed automatically in the right directory.

Documents Attachment Automation

Allows you to create rule templates to create attachment rules. When a new record is created, the corresponding rules and directories are then created.

Documents Archive

Allows to download directories as a archive file. It also possible to download multiple files at once, in that case the files are also packed together into an archive file.
